HealthyNest is a modern, high-performance food ordering web application designed to deliver a smooth and engaging user experience.
It allows users to explore healthy meals, customize their orders, and send them directly via WhatsApp for quick and easy checkout.

🔗 Live Demo: https://healthynest.vercel.app

🚀 Features
🧩 Build Your Meal (Custom Order System)
Users can fully customize their meals (e.g. sandwiches, salads) with different ingredients and options.
🛒 Smart Cart System
Add multiple items (custom or fixed) to the cart with real-time updates.
📲 WhatsApp Order Integration
On checkout, all cart data is automatically formatted and sent as a message via WhatsApp.
🎯 Modern UI/UX
Clean, attractive, and conversion-focused design with smooth animations.
⚡ High Performance
Optimized for speed and responsiveness across all devices.
📱 Fully Responsive
Works perfectly on mobile, tablet, and desktop.
🛠️ Tech Stack
Frontend: React / Next.js
Styling: Tailwind CSS
Animations: Framer Motion
Language: TypeScript
Deployment: Vercel
💡 Project Idea

Most restaurant websites are limited to just displaying a menu.
HealthyNest solves this by turning the website into an interactive ordering experience, making it easier for users to:

Explore meals visually
Customize their food
Order instantly without friction
